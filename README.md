## Data Availability
**NOTE:** Data will be released upon manuscript acceptance.

Raw data (fastq files) and aligned data (bam files) are available upon request at the EGA study [EGAS50000001612](https://www.ega-archive.org/studies/EGAS50000001612).  

All the processed data (bigwis, counts, etc.) used in all the downstream analyses are publicly available at GEO SuperSeries: [GSE320196](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE320196).
